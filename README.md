# Memory Sleuth

A guided setup wizard for onboarding and trial conversion flow.

## Usage

1. Run `python src/memory_sleuth.py` to start the setup wizard.
2. Follow the prompts to install the agent, generate an API key, and select feature flags.
3. The progress will be saved locally and can be resumed later.
4. Once the setup wizard is completed, a welcome email will be sent with next steps.
